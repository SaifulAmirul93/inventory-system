About Invento

Invento is an Inventory System that will allow you to keep control of the available items in your warehouse or business in general. Invento has the following features:

  • Order your items with categories
  • Create new items with a description, quantity and price
  • Create new categories with name, place and description
  • Statistics about your warehouse by week, month, year, or all time
  • 4 different user roles for new users (Administrator, General Supervisor, Supervisor and Employee)
  • Logs of each check-in, check-out or price change
  • Fully responsive design

Installation

To install Invento, follow the next steps:

  • Open the .zip file you've downloaded from CodeCanyon
  • Open the Invento/ folder and extract all the files.
  • With your preferred text editor, open the config.php file and edit the first 4 variables with your MySQL Details. Read the comments.
  • Upload all the files to your server
  • Open the install.php site on your webpage. e.g. If you uploaded the files to http://mysite.com/invento, go to http://mysite.com/invento/install.php. Invento will be installed automatically
  • Once Invento is installed, write down your username/password and delete the install.php file.
  • Log into your Invento account and change your password immediately

Updating from older version

To update Invento from an older version, please follow the steps below:

  • Open the .zip file you've downloaded from CodeCanyon
  • Open the Invento/ folder and extract all the files.
  • With your preferred text editor, open the config.php file and edit the first 4 variables with your MySQL Details. Read the comments.
  • Upload all the files to your server, replacing the old ones.
  • If you want to ensure a proper Invento installation, to do the 4th step, first delete all the old Invento files from your server and then upload the new files. Don't worry, nothing in your database will be deleted.
  • Done!

First Steps

As first steps, you may want to add new users or create new items.

If you want to add new users, just go to the Users section on the top right zone (after logging-in) and click the New User button. To check out the different User Roles and their permissions, please click here

If you want to create a new item, the first thing you need to do is to create a new category. To do this, just go to the Categories section and click the New Category button. After creating it, you'll be able to add new items.
